📊 Data Analytics Project – Customer Behavior Analysis
📌 Overview

This project focuses on analyzing customer behavior using real-world data to extract actionable business insights. It involves data cleaning, exploratory data analysis (EDA), SQL-based analysis, and visualization using Power BI. The final output is a comprehensive dashboard and a presentation summarizing key findings for decision-making.

📁 Dataset

The dataset contains customer transaction and behavior-related information such as:

Customer ID
Product / Item Purchased
Purchase Date
Quantity and Price
Review Ratings
Customer Demographics (if available)

The dataset is loaded and processed using Python for initial analysis and cleaning.
